# Agent Memory 专题

Agent 的记忆机制是实现长期对话、知识积累和个性化交互的核心技术。

## 核心方向

- **生物启发架构** - 人类海马体、情景记忆机制
- **无限/极长上下文** - 突破固定上下文窗口
- **分层/模块化记忆** - 工作记忆、长期记忆、情景记忆分离
- **动态知识更新** - 无需重训练的实时记忆修改
- **多Agent协作记忆** - 跨Agent的记忆共享与同步
