# 论文显微镜

这里记录我对 AI Agent 相关论文的精读和复现尝试。

## 🎯 阅读理念

> **"Read papers like a scientist, reproduce like an engineer."**

论文阅读不只是理解内容，更要：
1. **找到核心贡献** - 这篇论文解决了什么问题？
2. **理解方法** - 如何解决？关键思路是什么？
3. **评估实验** - 实验设计是否合理？结果是否可信？
4. **尝试复现** - 能跑通代码吗？能复现结果吗？
5. **提出思考** - 有什么局限性？可以如何改进？

## 📖 阅读方法论

[查看详细的论文阅读方法论 →](/zh/papers/reading-method)

### 快速阅读（15分钟）

适用于筛选论文：
1. 读标题、摘要、结论
2. 看图和表
3. 判断是否需要深度阅读

### 深度阅读（2-4小时）

适用于核心论文：
1. 完整阅读，做笔记
2. 理解公式和算法
3. 分析实验设计
4. 思考局限性和扩展方向

### 复现级阅读（1-2周）

适用于关键论文：
1. 找到官方代码
2. 跑通 baseline
3. 理解每行关键代码
4. 尝试修改和扩展
