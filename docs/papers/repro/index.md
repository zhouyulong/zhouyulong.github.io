## 复现方法论

1. **理解论文核心贡献**
2. **找到官方代码或第三方实现**
3. **跑通 baseline**
4. **理解关键代码**
5. **尝试改进或扩展**
6. **记录问题和解决方案**

## 复现模板

每个复现项目包含：
- `README.md` - 项目说明
- `notes/` - 复现笔记
- `code/` - 代码（如适用）
