# 关于我

## 简介

从事 **AI 算力平台**相关工作，关注 **AI Agent** 技术发展与研究。

希望从工程实现和系统架构的视角，深入理解智能体技术，探索从算力基础设施到上层应用的全栈优化。

## 关注方向

| 方向 | 兴趣点 | 当前状态 |
|------|--------|----------|
| **Multi-Agent System** | 协作机制、涌现行为、通信优化 | 学习中 |
| **Agent 推理效率** | 长上下文优化、推理加速、成本分析 | 结合现有经验 |
| **Agent 安全与对齐** | 可解释性、可控性 | 初步了解 |

## 这个网站的目的

1. **学习记录**：系统整理 Agent 领域的知识
2. **技术分享**：与同行交流，建立技术网络
3. **思考沉淀**：记录想法，追踪认知变化

## 内容特色

- **源码级分析**：不只是用框架，更要理解原理
- **复现驱动**：论文要读，代码要跑，坑要踩
- **开放思考**：提出开放性问题，记录研究想法
- **基础设施视角**：从算力成本、系统优化角度看 Agent
