## 什么内容会放入这里？

✅ **适合放入**：
- **系统性调研** - 某一方向的文献综述（如 Agent Memory 调研）
- **深度笔记** - 阅读多篇论文后的主题性总结
- **方法沉淀** - 经过验证的研究方法、工作流程

❌ **不适合放入**：
- 单篇论文的阅读笔记（放[论文中心](../papers/)）
- 临时想法（放[博客](../blog/)）
- 待办事项（放[待读池](../papers/todo-queue)）

---

## 专题扩展方式

新增技术专题时：

1. 创建 `docs/knowledge/topics/{topic-name}/index.md`
2. 在本文档「技术专题」表格中添加链接
3. 开展调研后，在专题目录下添加 `survey-YYYY-MM.md`
4. 阅读论文后，在专题目录下添加深度笔记
