# 知识体系

这里存放**长期积累的深度内容**，是经过整理、反复验证的知识体系。

## 包含内容

| 分类 | 说明 | 链接 |
|------|------|------|
| **方法论** | 如何高效阅读论文、做研究 | [论文阅读方法论](./paper-reading-method/) |
| **技术体系** | Agent 技术的系统梳理 | [Agent 技术演化](./agent-evolution/) |

## 什么内容会放入这里？

✅ **适合放入**：
- 经过验证的方法论
- 系统性的技术梳理
- 长期有效的知识体系

❌ **不适合放入**：
- 临时笔记（放博客）
- 待办事项（放论文-待读池）
- 实验性想法（放实验室）

---

*知识体系的建立是一个长期过程， slowly but steadily.*
