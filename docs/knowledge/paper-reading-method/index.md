# 论文阅读方法论

如何高效阅读 AI 论文？

## 三层阅读法

### 第一层：快速浏览（15分钟）

目标：判断论文是否值得深入阅读

- 读标题、摘要、结论
- 看图和表
- 扫读实验结果

### 第二层：深度理解（2-4小时）

目标：理解论文的核心贡献和方法

- 完整阅读引言，理解问题背景
- 仔细阅读方法部分，理解技术细节
- 分析实验设计是否合理
- 思考局限性和改进方向

### 第三层：批判思考（持续）

目标：形成自己的见解

- 这篇论文解决了什么问题？
- 方法的核心 insight 是什么？
- 实验是否充分支持结论？
- 有什么可以改进的地方？
- 对我的研究有什么启发？
