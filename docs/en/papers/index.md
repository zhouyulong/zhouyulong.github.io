# Paper Microscope

This is where I document my in-depth reading and reproduction attempts of AI Agent related papers.

## 🎯 Reading Philosophy

> **"Read papers like a scientist, reproduce like an engineer."**

Reading papers is not just about understanding the content, but also:
1. **Identify Core Contribution** - What problem does this paper solve?
2. **Understand the Method** - How is it solved? What are the key insights?
3. **Evaluate Experiments** - Is the experimental design sound? Are results credible?
4. **Attempt Reproduction** - Can the code run? Can results be reproduced?
5. **Propose Thoughts** - What are the limitations? How can it be improved?

## 📖 Reading Methodology

[View detailed paper reading methodology →](/en/papers/reading-method)

### Quick Reading (15 minutes)

For paper screening:
1. Read title, abstract, conclusion
2. Look at figures and tables
3. Decide if deep reading is needed

### Deep Reading (2-4 hours)

For core papers:
1. Read thoroughly, take notes
2. Understand formulas and algorithms
3. Analyze experimental design
4. Think about limitations and extensions

### Reproduction-Level Reading (1-2 weeks)

For key papers:
1. Find official code
2. Run baseline
3. Understand every line of key code
4. Try modifications and extensions
