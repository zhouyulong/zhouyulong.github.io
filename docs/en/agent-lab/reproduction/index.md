## Reproduction Methodology

1. **Understand the Paper** - Identify core contributions and innovations
2. **Find Code** - Prefer official implementations
3. **Environment Setup** - Document all dependencies and versions
4. **Run Baseline** - Make it work first, understand later
5. **Deep Analysis** - Understand key code logic
6. **Try Improvements** - Propose your own improvements
