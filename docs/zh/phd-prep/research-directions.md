# 研究方向探索

分析和比较不同的研究方向，为读博选择做准备。

## 评估维度

- **个人兴趣** - 是否真正感兴趣
- **能力匹配** - 是否发挥现有优势
- **发展前景** - 学术界和工业界前景
- **竞争激烈程度** - 申请难度
- **导师资源** - 该方向的优质导师数量
