# 学习记录模板

## 版本选择

根据阅读阶段和目的，提供三个版本：
- **快速版**：30分钟完成，适合泛读记录
- **标准版**：2-3小时完成，适合精读核心论文
- **深度版**：1-2天完成，适合展示的思考笔记

## 使用建议

### 什么时候用什么版本？

| 场景 | 推荐版本 | 预计时间 |
|------|---------|---------|
| 第一次通读，建立印象 | 快速版 | 30分钟 |
| 核心论文，需要深入理解 | 标准版 | 2-3小时 |
| 准备发给导师看 | 深度版 | 1-2天 |
| 写进申请文书 | 深度版的"与我工作的结合"部分 | - |

### 记录原则

1. **先完成再完美**：先用快速版记录，有时间再补充
2. **强制输出**：哪怕理解不深，也要强迫自己写总结
3. **链接思维**：每篇笔记都要关联到其他笔记或问题
4. **定期回顾**：每周花30分钟回顾本周笔记，更新理解

### 文件命名建议

```
YYYY-MM-DD_作者_标题关键词_版本.md

例如：
2026-04-01_Smith_AgentMemorySurvey_v1.md
```
